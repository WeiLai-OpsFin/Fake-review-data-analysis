# Data Dictionary

## Questionnaire Constructs

| Construct | Meaning |
|---|---|
| `ReviewType` | Experimental condition indicating real-review or fake-review exposure |
| `PreBookingIntention` | Baseline hotel booking intention before review exposure |
| `ReviewAuthenticity` | Respondent perception that the review is authentic and credible |
| `Trust` | Trust generated by the review information |
| `PerceivedRisk` | Perceived uncertainty or risk in making the booking decision |
| `BookingIntention` | Post-exposure hotel booking intention |

## Result Files

| File | Description |
|---|---|
| `results/did_summary.csv` | DID estimate and confidence interval summary |
| `results/did_regression.csv` | Regression-style DID output |
| `results/did_group_means.csv` | Group means before and after review exposure |
| `results/condition_comparison.csv` | Real-vs-fake construct mean comparisons |
| `results/measurement_model.csv` | Reliability, composite reliability, and AVE checks |
| `results/structural_paths_bootstrap.csv` | Bootstrapped structural path estimates |
| `results/mediation_effects.csv` | Indirect-effect and mediation tests |
| `results/r2_q2.csv` | R2, adjusted R2, Q2, RMSE, and MAE |
| `results/ipma_table.csv` | Importance-performance analysis for booking intention |

## Data Files

| File | Description |
|---|---|
| `data/pls_sem_construct_scores.csv` | Standardized construct scores used for the SEM-style analysis |
| `data/review_examples_20.csv` | Small public/constructed review-example set for demonstration |
